Это веб-плеер, который я сделал, чтобы разобраться с организацией данных в JavaScript. В нём можно просматривать альбомы, выбирать треки и слушать музыку прямо в браузере. Интерфейс стилизован под музыкальное приложение с карточками альбомов и списком треков.

Проект построен на нативном JavaScript, без сторонних библиотек. В коде используются классы для описания треков, альбомов и плейлистов. Треки и альбомы связываются между собой через единое хранилище, что позволяет легко находить композиции и добавлять их в разные подборки. Примеры треков взяты из открытых источников (Free Music Archive) и распространяются по лицензии CC BY.

Для демонстрации добавлены три композиции с соответствующими обложками. Плеер умеет запускать и останавливать воспроизведение, переключать треки и показывает информацию о текущей песне. Все данные о треках и альбомах хранятся в памяти приложения, так что при перезагрузке страницы структура не сбивается.

Чтобы запустить проект, достаточно скачать репозиторий и открыть файл html_code/music_app_library.html в браузере. Дополнительные зависимости не требуются.

Из того, что можно доработать в будущем: добавить очередь воспроизведения, регулировку громкости и сохранение текущего трека в localStorage, чтобы плеер запоминал, на чём остановился.
